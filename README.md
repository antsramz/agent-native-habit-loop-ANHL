# Agent-Native Habit Loop (ANHL)
A machine-only habit loop enabling autonomous agents to form, reinforce, and adjust behaviors using deterministic trigger-action-reward cycles.

---

## What this template does
ANHL gives agents a structured way to build habits, adapt them over time, and reinforce behaviors without human-readable layers.

---

## How it works
1. Agent detects a trigger.  
2. Agent performs the associated action.  
3. Reward signal reinforces or weakens the loop.  
4. Adjustment rules modify the habit over time.  
5. A machine-readable summary is returned.
